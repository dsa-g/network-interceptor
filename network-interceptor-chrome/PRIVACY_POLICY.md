# Privacy Policy

## Network Editor & Resend Chrome Extension

### Data Collection
This extension does **not** collect, store, or transmit any user data to external servers. All network request data is processed locally within your browser.

### Permissions Used
- **debugger**: Required to capture network requests through Chrome DevTools API
- **storage**: Used to save user preferences locally

### How Your Data is Used
- Network requests are captured locally and displayed in the DevTools panel
- Modified requests are sent directly from your browser to the target servers
- No data is sent to any third-party servers

### Data Storage
- Request history is stored temporarily in browser memory and cleared when DevTools is closed
- No persistent storage of network request data

### Third-Party Access
This extension does not share any data with third parties.

### Contact
For questions about this privacy policy, please open an issue on our GitHub repository: https://github.com/dsa-g/network-interceptor
