# Network Editor & Resend - Edge

Microsoft Edge DevTools extension to edit and resend network requests.

## Installation

1. Download `network-interceptor-edge.zip` and extract
2. Go to `edge://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extracted folder

## Usage

1. Open DevTools (F12)
2. Go to "Edit & Resend" tab
3. Click any request to edit and resend

## Publish to Edge Add-ons (Free!)

1. Go to https://partner.microsoft.com/dashboard
2. Create a free account
3. Submit the extension for review
4. Once approved, users can install from Edge Add-ons store

## Features

- Captures all network requests via Edge DevTools API
- Edit URL, HTTP method, headers, and request body
- Resend modified requests
- View response status, headers, and body
- Filter requests by URL or method
